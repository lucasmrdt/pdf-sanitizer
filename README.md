## Installation

```bash
git clone https://github.com/lucasmrdt/sanatize-pdf
cd sanatize-pdf
pip install -r requirements.txt --user
```

## Usage

```bash
> ./sanatize-pdf input_file.pdf output_file.pdf
```

## Example

```bash
> ./sanatize-pdf my_huge_file.pdf my_readable_file.pdf
```
